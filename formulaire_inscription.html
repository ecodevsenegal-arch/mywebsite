Étape 1 — Intégrer le formulaire dans votre site web

3 méthodes selon votre type de site (WordPress, HTML pur, CMS)

Méthode A — iFrame (la plus simple, fonctionne partout)

HTML 
<!-- Coller dans votre page HTML ou éditeur WordPress -->
<iframe
  src="formulaire_ecodev.html"
  width="100%"
  height="900"
  style="border:none; border-radius:12px;"
  title="Formulaire d'inscription ECODEV">
</iframe>

Méthode B — Inclusion directe dans une page HTML

HTML 
<!-- Dans votre page principale, remplacez le <body> -->
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>ECODEV — Inscription</title>
  <!-- Coller le contenu <style> du formulaire ici -->
</head>
<body>
  <!-- Coller le contenu <body> du formulaire ici -->
  <script src="form-logic.js"></script>
</body>
</html>

Méthode C — WordPress (bloc HTML personnalisé)

  • 1 Dans WordPress, ouvrez l'éditeur Gutenberg
  • 2 Ajoutez un bloc "HTML personnalisé"
  • 3 Collez le code iFrame de la Méthode A
  • 4 Hébergez le fichier formulaire_ecodev.html dans le dossier /wp-content/uploads/
  • 5 Ajustez l'URL src="" avec le chemin complet vers votre fichier
💡 Pour les données de production, remplacez l'action du formulaire par une URL de traitement serveur (voir Étape 3 pour Formspree ou EmailJS).

Étape 2 — Recevoir les données (sans serveur)

Utilisez Formspree ou EmailJS pour collecter les inscriptions par email

Option A — Formspree (gratuit, 50 soumissions/mois)

  • 1 Créez un compte sur formspree.io
  • 2 Créez un nouveau formulaire → copiez votre endpoint unique
  • 3 Modifiez la balise <form> dans le fichier HTML :

      action="https://formspree.io/f/VOTRE_ID"
      method="POST">

Étape 3 — Importer dans Google Forms

Recréer le formulaire dans Google Forms et exporter les réponses

Google Forms ne permet pas d'importer directement un fichier HTML. Voici le mapping complet de tous les champs :

🟢 Champs à créer dans Google Forms

  • Titre → Choix multiples (Madame / Monsieur)
  • Prénom → Réponse courte
  • Nom de famille → Réponse courte
  • Structure / Service → Réponse courte
  • Fonction → Réponse courte
  • Email → Réponse courte (validation email)
  • Adresse → Réponse courte
  • Ville → Réponse courte
  • Pays → Liste déroulante (58 pays)
  • Autre pays → Réponse courte
  • Téléphone → Réponse courte
  • Formation → Liste déroulante
  • Autre formation → Réponse courte
  • Mode de participation → Choix multiples

📋 Procédure étape par étape

  • Aller sur forms.google.com
  • Cliquer "+ Nouveau formulaire"
  • Ajouter le titre : "Inscription Formation ECODEV"
  • Ajouter chaque champ selon le mapping
  • Activer "Obligatoire" sur les champs requis
  • Utiliser "Logique de section" pour pays=Autre
  • Cliquer Envoyer → Lien à partager
  • Ou Envoyer → Intégrer pour obtenir l'iFrame
Script Google Apps (automatisation) 
// Coller dans Extensions → Apps Script du Google Form
// Pour envoyer une confirmation email à chaque inscription

function onFormSubmit(e) {
  var reponses = e.response.getItemResponses();
  var email = "";
  var nom = "";
  var formation = "";
  
  reponses.forEach(function(r) {
    if (r.getItem().getTitle() === "Email") email = r.getResponse();
    if (r.getItem().getTitle() === "Nom de famille") nom = r.getResponse();
    if (r.getItem().getTitle() === "Formation") formation = r.getResponse();
  });
  
  if (email) {
    MailApp.sendEmail({
      to: email,
      subject: "✅ Confirmation d'inscription — ECODEV",
      body: "Bonjour " + nom + ",\n\n" +
            "Votre inscription à la formation « " + formation + " » " +
            "a bien été reçue.\n\n" +
            "L'équipe ECODEV vous contactera prochainement.\n\n" +
            "Cordialement,\nECODEV Centre de Formation"
    });
  }
}

// Lier ce script au déclencheur "À l'envoi du formulaire"

Étape 4 — Télécharger les données sous Excel

3 méthodes pour exporter les inscriptions en fichier .xlsx

✅ Méthode 1 — Via Google Forms (la plus simple)

  • 1 Ouvrez votre Google Form → onglet Réponses
  • 2 Cliquez le bouton Google Sheets (icône tableau) pour créer une feuille liée
  • 3 Dans la Google Sheet → Fichier → Télécharger → Microsoft Excel (.xlsx)
  • 4 Le fichier contient toutes les inscriptions avec date et heure automatiques
⚡ La Google Sheet se met à jour automatiquement à chaque nouvelle inscription. Il suffit de la télécharger quand vous en avez besoin.

✅ Méthode 2 — Via Formspree Dashboard

  • 1 Connectez-vous à formspree.io → votre formulaire
  • 2 Onglet "Submissions" → bouton "Export CSV"
  • 3 Ouvrez le CSV dans Excel → Fichier → Enregistrer sous → Format Excel

✅ Méthode 3 — Script Apps Script automatique (envoi hebdomadaire)

// Exporter automatiquement vers Excel chaque lundi
// Coller dans Extensions → Apps Script de la Google Sheet

function exportWeeklyExcel() {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheet = ss.getActiveSheet();
  var data = sheet.getDataRange().getValues();
  
  // Créer un fichier Excel temporaire
  var blob = ss.getBlob().setContentType(
    'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet'
  );
  
  // Envoyer par email
  MailApp.sendEmail({
    to: "votre-email@domaine.com",
    subject: "📊 Export Inscriptions ECODEV — " + new Date().toLocaleDateString('fr-FR'),
    body: "Veuillez trouver en pièce jointe l'export des inscriptions.",
    attachments: [blob]
  });
}

// Créer un déclencheur temporel :
// Extensions → Apps Script → Déclencheurs → Ajouter
// Choisir : exportWeeklyExcel / Hebdomadaire / Lundi / 8h-9h
